# AI_QuestionAnswering
本人在项目中主要负责核心功能开发、系统集成与环境搭建工作，具体分工如下： 1.服务器租用与运行环境配置，保障系统稳定部署与访问。 2.关键词摘要功能的设计与集成，实现文本核心信息的高效提取。 3.情感分析与文本分类功能的外部调用接口整合（app.py 中核心逻辑开发）。 4.分析结果的统一 HTML 格式化展示设计（format_analysis 函数实现）。 5.sentiment_analysis.py 和 text_categorization.py 中 predict 函数的外部调用封装。 6.分析结果与主回复的拼接逻辑开发，确保多模块结果的协调展示。
